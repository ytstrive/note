### 目录

### 常用命令

#### 查看文件大小

#du -sh abc.txt

#du -sh web

#du -shc \*.css

**常用参数**

- -s 当前目录文件大小
- -h 以 k、m、g 为单位
- -c 或-total 除显示目录文件大小之外,同时显示总和

#### 查看进程

#ps aux | grep node

**常用参数**

- a 显示所有程序包含其它用户的程序
- u 按照用户排序显示
- x 显示所有程序
- grep 查找特定程序 如 node

#### 后台运行

#nohup node servier.js &

**常用参数**

- node 以 node 方式启动 servier.js
- & 放在命令最后表示以后台运行

#### 查看安装路径

#which node

**常用参数**

- node 文件名
